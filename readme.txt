========================================================Environment and libraries setup============================================================================
The following python libraries are needed to run the respective files:
1.nltk
2.sklearn
3.mathplotlib
4.pandas
5.numpy
6.time
7.pycm
8.re
9.bs4
10.urllib

========================================================Data Crawling==============================================================================================
python pip 
1.Open up a cmd terminal and cd to python bin folder
2.Run the restaurant_crawler.py/reviews_crawler.py file using python restaurant_crawler on your terminal
3.Change the url variable in each of the python files to crawl restaurants or reviews from a different page

========================================================Data cleaning==============================================================================================
1. Run the data_cleaning.py file
2. Change the csv file on the df variable to clean data for different files 

========================================================Data Sampling==============================================================================================
1.Run sampling_data.py file
2. Change the csv to sample data for different csv files

========================================================Model Training=============================================================================================
1. Execute the IR-ML.py file, this is the file to see the results of various model performance
2. Change the Corpus data to the evaluation data, labelled_data.csv and the test_data to reviews_uncleaned to obtain the auto classification results.
